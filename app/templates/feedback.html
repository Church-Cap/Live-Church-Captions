Diagnostics for support

Diagnostics are available for macOS, Windows, and Linux. They can include Church Cap version, OS version, CPU details, memory size, project drive capacity/free space, Python version, performance settings, CUDA/Apple runtime status, recent runtime metrics, and recent updater/CUDA logs with local paths redacted.

Review before sharing.

Diagnostics do not include transcripts, passwords, secrets, or .env contents, but system names, device names, error messages, and log details may be sensitive to your church or computer. Do not post the file publicly on GitHub unless you have reviewed it and are comfortable sharing it.
