Mode: {{ runtime.translation_provider or translation_provider }}

Important: phone UI translation and caption translation are separate. Whisper transcribes speech. Translated captions require an extra local translation provider.

Performance safeguard: each active translated language uses extra CPU/RAM. Church Cap automatically translates the most-requested languages up to the limit below. Others receive source captions with a help message.

Support

Diagnostics export

Mac / Windows / Linux

Download a diagnostics JSON file for troubleshooting or support. It can include Church Cap version, OS version, CPU details, memory size, project drive capacity/free space, Python version, performance settings, CUDA/Apple runtime status, recent runtime metrics, and updater/CUDA logs with local paths redacted.

Review before sharing.

Diagnostics do not include transcripts, captions, passwords, secrets, or .env contents, but system names, device names, error messages, and log details may still identify your church or computer. Do not post the file publicly on GitHub unless you have reviewed it and are comfortable sharing it.
