Recommended OBS settings

  1. In OBS, click Sources → + → Browser.
  2. Name the source Live Captions.
  3. Paste the direct IP browser-source URL above. It normally ends in /obs.
  4. Set width to 1920 and height to 300 for a lower-third layout.
  5. Place the source near the bottom of the stream canvas.
  6. Enable Refresh browser when scene becomes active if you want OBS to reconnect automatically when switching scenes.

Operating notes

The overlay has a transparent background and shows the current caption with the same smoother caption pacing used by the audience display. If the overlay is blank, confirm captions are running, the OBS computer can reach the URL above, and no firewall is blocking the Church Cap port.

For public livestreams, remember that AI captions and translations can be inaccurate. They may also include spoken names, pastoral details, testimony, or copyrighted worship lyrics. Use blank/pause mode when captions should not be shown or retained.
