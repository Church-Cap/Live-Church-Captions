OBS browser-source setup

This is not a native OBS plugin yet. It is a browser-source overlay, which is simpler, local-first, and works well for livestream captions.

Recommended OBS settings

  1. In OBS, click Sources → + → Browser.
  2. Name it Live Captions.
  3. Set the URL to the browser source URL above.
  4. Set width to 1920 and height to 300 for a lower-third layout.
  5. Place it near the bottom of the stream canvas.
  6. Tick Refresh browser when scene becomes active if you want captions to reconnect each time.

Notes

The overlay has a transparent background and only displays the current caption. It uses the same local WebSocket broadcast as the phone and display pages.

For public livestreams, remember that AI captions may be inaccurate and may capture copyrighted worship lyrics or sensitive pastoral content. Use blank/pause mode when needed.
